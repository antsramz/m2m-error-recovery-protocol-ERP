# M2M Error Recovery Protocol (ERP)
A machine-only fallback and recovery system for handling failures, retries, degraded modes, and escalation paths between autonomous agents.

---

## What this template does
ERP provides a deterministic error-handling framework that agents can use to recover from failures without human intervention.

---

## How it works
1. Error is classified.  
2. Agent retries the operation based on policy.  
3. If retries fail, agent enters a degraded mode.  
4. If conditions require it, error is escalated.  
5. A machine-readable recovery summary is returned.
